2.4 转译机制

可以使用show_query函数来调查tidyverse包是如何把相关代码转译为SQL语句的。

mydb_products %>% 
  select(prod_name) %>% 
  show_query()
<SQL>
SELECT `prod_name`
FROM `Products`
